# okto  

[![forthebadge](http://forthebadge.com/images/badges/certified-snoop-lion.svg)](http://forthebadge.com)

An (impolite) sitemap builder

### Building

```
go get github.com/evgorchakov/okto
go build
```

### Running
```
okto --timeout=10s --rate=5 http://example.com
```

For more options:
```
okto --help
```
